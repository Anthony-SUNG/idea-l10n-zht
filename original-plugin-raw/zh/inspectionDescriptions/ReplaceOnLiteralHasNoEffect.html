Reports calls to the string methods replace(), replaceAll() or replaceFirst() that have no effect. Such calls can be guaranteed to have no effect when the qualifier and search string are compile-time constants and the search string is not found in the qualifier. 这是冗余代码，可能表示存在编程错误。

示例：


  // replacement does nothing
  "hello".replace("$value$", value);

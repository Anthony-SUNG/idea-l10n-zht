报告具有 @targetName 注解的定义，该注解覆盖了没有 @targetName 注解的定义。

请参阅 @targetName 注释。

该快速修复会移除 @targetName 注解。

示例：


  class A:
    def foo: Int = 1

  class B extends A:
    @targetName("overriddenFoo")
    override def foo: Int = 2

在应用快速修复后：


  class A:
    def foo: Int = 1

  class B extends A:
    override def foo: Int = 2

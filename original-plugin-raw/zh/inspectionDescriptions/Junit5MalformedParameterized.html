报告源格式错误的形参化测试：

  • MethodSource 具有未知目标或方法并非 static、no-arg。
  • ValueSource 和 EnumSource 类型不能转换为方法形参。
  • 未定义来源。

示例：


  class Test {
    @MethodSource("parameters")
    @ParameterizedTest
    void foo(String param) {}
  }

在应用快速修复后：


  class Test {
    private static Stream<Arguments> parameters() {
        return Stream.empty();
    }

    @MethodSource("parameters")
    @ParameterizedTest
    void foo(String param) {}
  }

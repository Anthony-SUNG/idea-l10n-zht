Reports classes without a constructor that takes no arguments (i.e. has no parameters). No-arg constructors are necessary in some contexts. 例如，如果需要使用反射来创建类。

示例：


  public class Bean {
    private String name;

    public Bean(String name) {
      this.name = name;
    }
  }

使用下面的复选框可忽略没有显式构造函数的类。 The compiler provides a default no-arg constructor to such classes.

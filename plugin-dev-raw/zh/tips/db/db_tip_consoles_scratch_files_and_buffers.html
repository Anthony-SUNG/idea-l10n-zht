數據庫控制台 是已附加到數據源的 SQL 檔案。創建數據源時，會自動創建數據庫控制台。要打開控制台，請右鍵點擊數據源並選擇 新建 | 查詢控制台。

臨時檔案 是未附加到數據源的 SQL 檔案。對於項目上下文之外的一些代碼想法，將臨時檔案稱為臨時註釋或草稿。要創建臨時檔案，請導航至 檔案 | 新建 | 臨時 (或按 &shortcut:NewScratchFile;)。

臨時緩衝區 是簡單的文本檔案，沒有任何編碼輔助功能。臨時緩衝區可用於簡單的任務列表和給自己的筆記。沒有專用的菜單項用於創建新的臨時緩衝區，但您可以使用 查找操作 彈出視窗(&shortcut:GotoAction; )並運行 新建臨時緩衝區 操作。

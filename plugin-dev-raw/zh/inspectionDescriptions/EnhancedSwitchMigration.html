報告可以自動替換為增強型 switch 語句或表達式的 switch 語句。

示例：


  double getPrice(String fruit) {
    // Switch 語句可以替換為增強型 'switch'
    switch (fruit) {
      case "Apple":
        return 1.0;
      case "Orange":
        return 1.5;
      case "Mango":
        return 2.0;
      default:
        throw new IllegalArgumentException();
    }
  }

在應用快速修復后：


  double getPrice(String fruit) {
    return switch (fruit) {
      case "Apple" -> 1.0;
      case "Orange" -> 1.5;
      case "Mango" -> 2.0;
      default -> throw new IllegalArgumentException();
    };
  }

此檢查僅適用于 14 級或更高的語言級別

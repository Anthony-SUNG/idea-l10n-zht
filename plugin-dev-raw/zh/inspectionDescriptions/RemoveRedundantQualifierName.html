報告關於類名、函數和屬性的冗餘限定符（或其部分）。

完全限定名稱是一種明確的標識符，用於指定調用所引用的對象、函數或屬性。 在可以縮短名稱的上下文中，檢查會告知機會，關聯的“移除冗餘限定符名稱”快速修復可以修改程式碼。

示例：


  package my.simple.name
  import kotlin.Int.Companion.MAX_VALUE

  class Foo

  fun main() {
      val a = my.simple.name.Foo()    // 'Foo' 位於聲明的 'my.simple.name' 軟件包中，限定符冗餘
      val b = kotlin.Int.MAX_VALUE    // 可以取代為 'MAX_VALUE'，因為它已被導入
      val c = kotlin.Double.MAX_VALUE // 可以取代為 'Double.MAX_VALUE'，因為內置類型會被自動導入
  }

在應用快速修復後：


  package my.simple.name
  import kotlin.Int.Companion.MAX_VALUE

  class Foo

  fun main() {
      val a = Foo()
      val b = MAX_VALUE
      val c = Double.MAX_VALUE
  }

報告對 testing.T.FailNow 和位於測試檔案 goroutine 中的類似方法的調用。

像 FailNow 這種方法會調用 runtime.Goexit，並停止調用 goroutine，而不是測試。 因此，必須僅從運行測試或基準的 goroutine 中調用。

有關 FailNow 的更多信息，請參見 golang.org 的 func (*T) FailNow。

示例：

func TestFoo(t *testing.T) {
  go func() {
    t.Fatal("oops") // 退出 goroutine，而不是 TestFoo
  }()
}

在應用替換為 'Error' 和 'return' 快速修復後：

func TestFoo(t *testing.T) {
  go func() {
    t.Error("oops")
    return
  }()
}

報告對使用 java.lang.Object 中預設實現的 function toString() { [native code] }() 的調用。

該預設實現很少是有意為之，可能是誤用。

此檢查將忽略對包含 java.lang.Object、接口或 abstract 類類型的對象的 function toString() { [native code] }() 調用。

範例：


  class Bar {
      void foo1(Bar bar) {
          String s = bar.function toString() { [native code] }(); // 警告
          /* ... */
      }

      void foo2(Object obj) {
          String s = obj.function toString() { [native code] }(); // 此處無警告
          /* ... */
      }
  }

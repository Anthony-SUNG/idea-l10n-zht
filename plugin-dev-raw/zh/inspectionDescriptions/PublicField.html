報告 public 字段。 不報告常量（標記 static 和 final 的字段）。

示例：


  class Main {
    public String name;
  }

在應用快速修復後：


  class Main {
    private String name;

    public String getName() {
        return name;
    }

    public void setName(String name) {
        this.name = name;
    }
  }

配置檢查：

  • 使用有註解時忽略列表可指定要忽略的註解。 該檢查將忽略包含其中任何註解的字段。
  • 使用忽略枚舉的 'public final' 字段選項，可以忽略 enum 類型的 public final 字段。

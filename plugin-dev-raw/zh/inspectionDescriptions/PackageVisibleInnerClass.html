報告在沒有任何訪問修飾符（也稱為 package-private）的情況下宣告的嵌套類。

範例：


  public class Outer {
    static class Nested {}  // 警告
    class Inner {}          // 警告
    enum Mode {}            // 警告取決於設定
    interface I {}          // 警告取決於設定
  }

配置檢查：

  • 使用忽略軟件包可見的內部枚舉選項可忽略 package-private 內部枚舉。
  • 使用忽略軟件包可見的內部接口選項可忽略 package-private 內部接口。

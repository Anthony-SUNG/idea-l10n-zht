報告對不存在或不可見的字段和方法的反射訪問。

示例：


  Field stringHashField() throws NoSuchFieldException {
    return String.class.getField("hash");
  }

在應用快速修復後：


  Field stringHashField() throws NoSuchFieldException {
    return String.class.getDeclaredField("hash");
  }

對於 final 類，很清楚該類中是否存在具有指定名稱的字段或方法。

對於非 final 類，子類中可能有使用該名稱的字段或方法，因此可能會出現誤報。 使用該項檢查的設置，可消除任何位置或特定類的此類誤報。

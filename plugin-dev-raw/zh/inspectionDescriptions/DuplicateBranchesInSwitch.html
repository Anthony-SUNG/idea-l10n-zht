報告在不同分支中包含相同代碼的 switch 語句或表達式，並建議合並重復的分支。

示例：


  switch (n) {
    case 1:
      System.out.println(n);
      break;
    case 2:
      System.out.println(n);
      break;
    default:
      System.out.println("default");
  }

可以替換為：


  switch (n) {
    case 1:
    case 2:
      System.out.println(n);
      break;
    default:
      System.out.println("default");
  }

報告檢查已聲明變量非 null 值的 assert 調用。

使用 !! or ?: 可使代碼更簡單。

該快速修復會將變量初始值設定項中的 assert 取代為 !! 或 ?: 運算符。

示例：


  fun foo(p: Array<String?>) {
      val v = p[0]
      assert(v != null, { "Should be not null" })
  }

在應用快速修復後：


  fun foo(p: Array<String?>) {
      val v = p[0] ?: error("Should be not null")
  }

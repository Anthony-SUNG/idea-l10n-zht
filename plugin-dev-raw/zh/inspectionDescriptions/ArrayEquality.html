報告用于測試數組相等性的運算符 == 和 !=。 在大多數情況下，如果原本就要測試數組內容的相等性，可以通過 java.util.Arrays.equals() 方法來完成。

建議通過快速修復將 == 替換為 java.util.Arrays.equals()。

示例：


  void foo(Object[] x, Object[] y) {
    boolean comparison = x == y;
  }

在應用快速修復后：


  void foo(Object[] x, Object[] y) {
    boolean comparison = Arrays.equals(x, y);
  }

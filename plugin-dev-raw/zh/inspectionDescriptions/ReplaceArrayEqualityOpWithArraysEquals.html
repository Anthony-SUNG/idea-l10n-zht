報告 == 或 != 運算符的用法，用於應取代為 contentEquals() 的數組。

== 和 != 運算符比較數組引用而不是它們的內容。

示例：

  fun test() {
      val a = arrayOf(1, 2, 3)
      val b = arrayOf(1, 2, 3)
      println(a == b) // 參考比較
  }

在應用快速修復後：

  fun test() {
      val a = arrayOf(1, 2, 3)
      val b = arrayOf(1, 2, 3)
      println(a.contentEquals(b))
  }

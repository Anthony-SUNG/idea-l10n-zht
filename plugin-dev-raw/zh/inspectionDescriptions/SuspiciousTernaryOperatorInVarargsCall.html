報告使用具有混合數組分支和非數組分支的三元運算符的 vararg 方法調用。

編譯時，兩個分支都被包裝在數組中。 結果，數組分支變成了二維數組，這可能說明有問題。

該快速修復會將非數組分支包裝在數組中以防止編譯器執行轉換。

示例：


    static void bar(boolean flag) {
        Object[] a = {1, 2};
        Object b = "hello";
        foo(flag ? a : b);
    }
    static void foo(Object... obj) {
    }

應用該快速修復后： 


    static void bar(boolean flag) {
        Object[] a = {1, 2};
        Object b = "hello";
        foo(flag ? a : new Object[]{b});
    }
    static void foo(Object... obj) {
    }

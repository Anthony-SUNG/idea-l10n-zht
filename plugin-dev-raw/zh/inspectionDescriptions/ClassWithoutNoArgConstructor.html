Reports classes without a constructor that takes no arguments (i.e. has no parameters). No-arg constructors are necessary in some contexts. 例如，如果需要使用反射來創建類。

範例：


  public class Bean {
    private String name;

    public Bean(String name) {
      this.name = name;
    }
  }

使用下面的複選框可忽略沒有顯式構造函數的類。 The compiler provides a default no-arg constructor to such classes.

報告被聲明為 java.lang.String 且重複追加的變量。 此類變量可以更有效地被聲明為 java.lang.StringBuffer 或 java.lang.StringBuilder。

示例：


  String s = "";
  for (int i = 0; i < names.length; i++) {
    String name = names[i] + (i == names.length - 1 ? "" : " ");
    s = s + name;
  }

此類循環可以取代為：


  StringBuilder s = new StringBuilder();
  for (int i = 0; i < names.length; i++) {
    String name = names[i] + (i == names.length - 1 ? "" : " ");
    s.append(name);
  }

或者甚至可以取代為：


  String s = String.join(" ", names);

使用該選項可使此檢查僅在變量被附加到循環中時報告。

如果发生 org.testng.TestNGException，則報告具有相同名稱的 TestNG 數據提供者。

示例：


  public class DuplicatedDataProviders {
    @DataProvider
    public Object[][] intTestData() { // 重復項 1
      return new Integer[][]{
        new Integer[]{1, 1},
      };
    }

    @DataProvider(name = "intTestData")
    public Object[][] someTestData() { // 重復項 2
      return new Integer[][]{
        new Integer[]{1, 1},
      };
    }

    @Test(dataProvider = "intTestData")
    public void testIsOK(Integer key, Integer value) {
      assertEquals(key, value);
    }
  }

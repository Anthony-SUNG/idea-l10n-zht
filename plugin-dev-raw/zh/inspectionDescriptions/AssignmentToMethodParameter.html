報告對方法形參的賦值或修改。

雖然有時是有意為之，但此結構可能令人困惑，因此在一些 Java 項目中被禁止使用。

該快速修復會添加新變量的聲明。

示例：


  void printTrimmed(String s) {
    s = s.trim();
    System.out.println(s);
  }

在應用快速修復后：


  void printTrimmed(String s) {
    String trimmed = s.trim();
    System.out.println(trimmed);
  }

使用如果賦值是原始形參的轉換，則忽略選項可忽略根據先前值修改形參值的賦值。

報告沒有形參子句並使用類似轉變器的名稱重寫 Java 方法的方法。

Scala 允許重寫沒有形參子句的無形參 Java 方法。 這主要是為了允許 Scala 代碼以與 Scala 訪問器方法相同的方式重寫 Java 訪問器方法。 即，不使用任何形參子句並遵守統一訪問原則，也就是說客戶端代碼不應受到將特性實現為字段或方法的決定的影響。

但是，對於具有副作用的方法，通常有一個空形參子句。 這將符合里氏替換原則，即當被重寫的方法有副作用時，重寫方法也必須被聲明為具有副作用的方法。

該快速修復會添加一個空形參子句。

示例：


  import java.util

  class Test extends util.ArrayList[String] {
    override def clear: Unit = ???
  }

在應用快速修復後：


  import java.util

  class Test extends util.ArrayList[String] {
    override def clear(): Unit = ???
  }

報告類外部方法中的 @classmethod 或 @staticmethod 裝飾器的用法。

示例：

class State(object):

    @classmethod
    def my_state(cls, name):
        cls.name = name


@classmethod
def change_state(self):
    pass

change_state 方法不應使用 @classmethod 裝飾器，或應將其移動到 state 類聲明。

如果應用移除裝飾器操作，代碼變為：

class State(object):

    @classmethod
    def my_state(cls, name):
        cls.name = name


def change_state(self):
    pass

報告引用擁有注釋的方法、擁有注釋的方法的 super 方法或包含注釋的類的 Javadoc @see、{@link} 和 {@linkplain} 標記。

此類鏈接不必要，可以通過此檢查的快速修復安全移除。 如果標記是唯一內容，快速修復將移除整個 Javadoc 注釋。

示例：


  class Example {
    /**
     * @see Example#method
     */
    public void method() { }
  }

在應用快速修復後：


class Example {
  public void method() { }
}

使用下面的復選框可以忽略指向 super 方法的內聯鏈接（{@link} 和 {@linkplain}）。 儘管 Javadoc 工具會自動添加指向所有 super 方法的鏈接，但有時在 Javadoc 注釋的文本中可能需要指向 super 方法的內聯鏈接。

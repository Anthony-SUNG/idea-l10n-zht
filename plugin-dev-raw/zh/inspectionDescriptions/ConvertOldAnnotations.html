報告 TestNG org.testng.annotations.Configuration 注解。

最好使用現代 @BeforeXXX/@AfterXXX 注解來配置測試套件。

示例：


  public class Test {
    @Configuration(beforeSuite = true, afterTest = true)
    public void afterBefore(){
    }
  }

在應用快速修復后：


  public class Test {
    @AfterTest()
    @BeforeSuite()
    public void afterBefore(){
    }
  }

報告匿名類中名稱與包含方法或 lambda 表達式的局部變數或參數相同的字段。

由於進行此類命名，在包含方法中有意使用同名變數或參數的情況下，您可能會誤用匿名類字段。

建議通過快速修復重命名該字段。

示例：


  class Test {
    public Test(String value) {
      Object foo = new Object() {
        private String value = "TEST";
        public void foo() {
          System.out.println(value); //訪問的是字段，而不是參數
        }
      };
    }
  }

報告使用沒有 N 前綴的國家字符的字符串文字。

如果沒有 N 前綴，字符串將被轉換為數據庫的預設代碼頁面。 此預設代碼頁面可能無法識別某些字符。 有關更多信息，請參閱 docs.microsoft.com 上的“nchar 和 nvarchar”(Transact-SQL) 頁面。

示例（Microsoft SQL 服務器）：

SELECT 'abcde' AS a;
SELECT N'abcde' AS b;
SELECT 'абвгд' AS c;
SELECT N'абвгд' AS d;

SELECT 'абвгд' AS c; 沒有 N 前綴，'абвгд' 部分將被高亮顯示。

報告在單個聲明中聲明的多個變量，並建議為每個變量創建單獨的聲明。

一些編碼標准禁止此類聲明。

示例：


  int x = 1, y = 2;

在應用快速修復后：


  int x = 1;
  int y = 2;

配置檢查：

  • 使用 忽略 'for' 循環聲明 選項可忽略在 'for' 循環語句的初始化中聲明的多個變量，例如： 
    
  for (int i = 0, max = list.size(); i > max; i++) {}
  • 使用 僅在單個聲明中對不同的數組維度发出警告 選項，以僅在單個聲明中聲明具有不同數組維度的變量時发出警告，例如： 
    
  String s = "", array[];

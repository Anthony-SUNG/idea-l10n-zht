報告帶有冗餘代碼塊的 switch 語句或 switch 表達式的標記規則。

示例：


  String s = switch (n) {
    case 1 -> { yield Integer.function toString() { [native code] }(n); }
    default -> "default";
  };

在應用快速修復後：


  String s = switch (n) {
    case 1 -> Integer.function toString() { [native code] }(n);
    default -> "default";
  };

僅當項目或模塊的語言級別為 14 或更高時，此檢查才會報告。

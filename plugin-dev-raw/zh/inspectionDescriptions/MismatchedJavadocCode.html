報告用英語編寫的與方法宣告衝突的方法規范部分。 這包括：
  • 方法被指定為返回 true 或 false，但其返回類型不是布爾。
  • 方法被指定為返回 null，但被註解為 @NotNull 或其返回類型為基元類型。
  • 方法被指定為返回列表，但其返回類型為集合或陣列。
  • 以此類推。

範例：


  /**
   * @如果找到使用者，則返回 true，否則返回 false
   */
  User findUser(String name);

請注意，誤報是可能的，因為此檢查會試圖解釋一種人類語言。 然而，如果檢查報告不正確，描述仍然可能令人困惑，應該覆寫。

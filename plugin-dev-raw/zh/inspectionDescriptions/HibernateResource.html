如果返回的 org.hibernate.Session 資源沒有被安全關閉，報告對 openSession() 方法的調用。

默認情況下，該檢查假定資源可以由名稱中帶有 'close' 或 'cleanup' 的任何方法關閉。

示例：


  void doHibernateQuery(SessionFactory factory) {
    Session session = factory.openSession(); //警告
    session.createQuery("...");
  }

使用以下選項配置檢查：

  • 是否允許在 try 塊內打開 org.hibernate.Session 資源。 這種樣式不太合意，因為它比在 try 塊前面打開資源更冗長。
  • 資源能否通過任何將資源作為實參傳遞的方法調用來關閉。

報告 require() 語句。 建議使用 import 語句將其轉換為 require() 調用。

啟用“使用‘全部修復’操作轉換內部範圍中的 require()”， 在使用“全部修復”操作時轉換嵌套函數和語句內部的所有 require() 調用。

請注意，將內部範圍中的 require() 語句轉換為 import 語句 可能會導致程式碼語義更改。 Import 語句是 static 模塊依賴項，它們獲得了提升， 這表明它們被移至當前模塊的頂部。 require() 調用動態載入模塊。 它們可以有條件地執行，其範圍由使用它們的表達式來定義。
清除“使用‘全部修復’操作轉換內部範圍中的 require()”復選框，以防止在使用“全部修復”操作時對這些複雜情況進行任何更改。

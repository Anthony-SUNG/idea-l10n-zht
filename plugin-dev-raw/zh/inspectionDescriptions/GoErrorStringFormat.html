報告錯誤字符串的格式問題。

示例：

err := fmt.Errorf("Cannot read the file!")
log.Printf("Reading %s: %v", file, err)

根據 golang.org 上的 Error Strings 頁面，錯誤字符串不應大寫或以標點符號結尾，因為它們可能會出現在其他上下文中。

要修復格式，可以應用修復錯誤字符串格式快速修復。 應用快速修復后，錯誤字符串將如下所示：

err := fmt.Errorf("cannot read the file")
log.Printf("Reading %s: %v", file, err)

報告循環中最后能夠到達的 continue 語句。 這些 continue 語句是不必要的語句，可以安全移除。

示例：


  for (String element: elements) {
    System.out.println();
    continue;
  }

在應用快速修復后：


  for (String element: elements) {
    System.out.println();
  }

該檢查不分析 JSP 文件。

使用在具有 'else' 分支的 'if' 語句的 then 分支中忽略選項可以忽略位于完整 if-else 語句的 then 分支中的 continue語句。

示例：


  for (String element: elements) {
    if(element.isEmpty()) {
      continue;
    } else {
      //...
    }
  }

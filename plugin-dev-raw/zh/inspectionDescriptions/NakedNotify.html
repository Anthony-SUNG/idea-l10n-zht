報告調用的未发生任何可檢測到的狀態變更的 Object.notify() 或 Object.notifyAll()。

Object.notify() 和 Object.notifyAll() 通常用于通知其他線程发生了狀態變更。 該狀態變更應发生在包含 Object.notify() 或 Object.notifyAll() 調用的同步上下文中，並且发生在調用之前。 盡管沒有此類狀態變更未必就不正確，但肯定值得檢查。

示例：


  synchronized (this) {
    notify();
  }
  // 無狀態變更
  synchronized (this) {
    notify(); // 告知可能冗余
  }

報告基于實參的冗余 let 調用。

當 lambda 形參僅用作調用表達式中的限定符時，let 冗余。

如果需要為限定表達式命名，請聲明一個局部變量。

示例：


  fun splitNumbers() {
      "1,2,3".let { it.split(',') }
  }

該快速修復會移除額外的 let() 調用：


  fun example() {
      "1,2,3".split(',')
  }

或者：


  fun splitNumbers() {
      val numbers = "1,2,3"
      numbers.split(',')
  }

報告與相應 super 方法的形參類型同名但屬于不同軟件包的子類方法的形參類型。 在這些情況下，子類方法不能重寫 super 方法。

示例：


public class A {
 public void method(Object o) {}
}

public class B extends A {
 public void method(Object o) {} // 對形參類型发出警告
 class Object {}
}

在應用快速修復后：


public class A {
 public void method(Object o) {}
}

public class B extends A {
 public void method(java.lang.Object o) {} // 新形參類型
 class Object {}
}

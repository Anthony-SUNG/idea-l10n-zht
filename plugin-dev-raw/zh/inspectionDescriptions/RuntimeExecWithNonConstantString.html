報告對 java.lang.Runtime.exec() 的調用，該調用會將動態構造的字符串作為要執行的命令。

構造的執行字符串是安全漏洞的常見來源。 預設情況下，此檢查會忽略編譯時常量。

示例：


  String i = getUserInput();
  Runtime runtime = Runtime.getRuntime();
  runtime.exec("foo" + i); // 報告警告

使用檢查設置可以將任何 static final 字段視為常量。 請小心，因為啟用該選項時，像下面這樣的字符串將被忽略： 


  static final String COMMAND = "ping " + getDomainFromUserInput() + "'";

報告可以替換為方法引用的 lambda。 方法引用比 lambda 更容易理解和閱讀，盡管它們通常取決于您的喜好。

示例：


  Runnable r = () -> System.out.println();

在應用快速修復后：


  Runnable r = System.out::println;

即使 lambda 不調用任何方法，該檢查也可能會建議方法引用，例如將 obj -> obj != null 替換為 Objects::nonNull。 使用設置 | 編輯器 | 代碼樣式 | Java | 代碼生成設置來配置特殊方法引用。

僅當項目或模塊的語言級別為 8 或更高時，此檢查才會報告。

建議將 for(Entry<?,?> entry : map.entrySet()) {...} 或 map.entrySet().forEach(entry -> ...) 替換為 map.forEach((key, value) -> ...)。

示例


  void print(Map<String, Integer> map) {
    map.entrySet().forEach(entry -> {
      String str = entry.getKey();
      System.out.println(str + ":" + entry.getValue());
    });
  }

在應用快速修復後：


  void print(Map<String, Integer> map) {
    map.forEach((str, value) -> System.out.println(str + ":" + value));
  }

啟用 不要報告循環選項後，只會報告 entrySet().forEach() 情況。 但是，該快速修復操作也可用於 for 循環。

僅當項目或模塊的語言級別為 8 或更高時，此檢查才會報告。

報告無意義的算術表達式。 此類表達式包括加 0、減 0、乘以 0 或 1 以及除以 1。

此類表達式可能是自動重構所致，不太可能是開发者的本意。

該快速修復會簡化此類表達式。

示例： 


  void f(int a) {
    int x = a - a;
    int y = a + 0;
    int res = x / x;
  }

在應用快速修復后：


  void f(int a) {
    int x = 0;
    int y = a;
    int res = 1;
  }

請注意，在極少數情況下，建議的替換可能不完全等同于所有可能輸入的原始代碼。 例如，該檢查會建議將 x / x 替換為 1。 但是，如果 x 為零，原始代碼將拋出 ArithmeticException 或導致 NaN。 另外，如果 x 為 NaN，則結果也是 NaN。 此類行為不太可能出于本意。

報告可以替換為 Collection.indices 或 for 循環內集合迭代的 until 和 rangeTo 運算符。

使用語法糖可使代碼更簡單。

該快速修復會將手動范圍替換為相應的構造。

示例：


  fun main(args: Array<String>) {
      for (index in 0..args.size - 1) {
          println(args[index])
      }
  }

在應用快速修復后：


  fun main(args: Array<String>) {
      for (element in args) {
          println(element)
      }
  }

報告不遵循命名慣例的屬性。

報告了以下屬性：

  • 頂級屬性
  • 對象和伴生對象中的屬性

您可以在檢查選項中指定所需的模式。

建議的命名慣例：必須以大寫字母開頭，使用駝峰式命名並且沒有下划線。

範例：


  // 頂級屬性
  val USER_NAME_FIELD = "UserName"
  // 包含對單例對象的引用的頂級屬性
  val PersonComparator: Comparator<Person> = /*...*/

  class Person {
    companion object {
      // 伴生對象中的屬性
      val NO_NAME = Person()
    }
  }

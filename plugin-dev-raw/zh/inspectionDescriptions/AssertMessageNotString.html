報告非 java.lang.String 類型的 assert 消息。

使用字符串可以提供更多信息，有助於診斷失敗或斷言原因。

示例：


  void foo(List<Integer> myList) {
    assert myList.isEmpty() : false;
  }

使用僅當 assert 消息為 'boolean' 或 'java.lang.Boolean' 時發出警告選項可以僅在 assert 消息類型為 boolean 或 java.lang.Boolean 時發出警告。 boolean 詳細消息不太可能提供有關斷言失敗的其他信息，可能是由於錯誤輸入 : 而不是 & 所致。

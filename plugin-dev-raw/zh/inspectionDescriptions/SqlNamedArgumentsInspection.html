報告在例程調用中不帶名稱使用的實參。 預設情況下，此檢查處於禁用狀態。

有關命名形參和未命名形參之間差異的詳細信息，請參閱 docs.microsoft.com 上的 Binding Parameters by Name (Named Parameters) 頁面。

示例（Microsoft SQL 服務器）：

CREATE FUNCTION foo(n INT, m INT) RETURNS INT AS
BEGIN
    RETURN n + m;
END;

CREATE PROCEDURE test AS
BEGIN
    foo n = 1, m = 2;

--- 以下調用缺少形參名稱，會被高亮顯示
    foo 1, 2;
END;

foo 1, 2; 調用中的形參 1, 2 突出顯示，因為它們缺少名稱。

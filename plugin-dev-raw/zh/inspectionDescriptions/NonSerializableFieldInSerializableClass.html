報告實現 java.io.Serializable 的類中的不可序列化字段。 如果對象被序列化，此類字段將導致執行時異常。

不報告被聲明為 transient 或 static 的字段，也不報告定義了 writeObject 方法的類的字段。

此檢查假定 java.util.Collection 和 java.util.Map 類型的字段為 Serializable，除非聲明它們的類型為非 Serializable。

示例：


  class NonSerializableClass {}

  public class SerializableClass implements Serializable {
      NonSerializableClass clazz; // //警告：可序列化類中存在不可序列化字段 'clazz'
      static NonSerializableClass staticClazz; // 無警告
  }

使用以下選項配置檢查：

  • 列出此檢查不應報告其繼承者的類。 這適用於從超類繼承 Serializable 但不用於序列化的類。
  • 列出會使檢查忽略註解字段的註解。
  • 是否忽略使用匿名類初始化的字段。

報告在對象構造期間對當前類的重寫方法的任何調用。 如果對象構造在內部，則會发生這種情況：
  • 構造函數
  • 非 static 實例初始值設定項
  • 非 static 字段初始值設定項
  • clone()
  • readObject()
  • readObjectNoData()

此類調用可能會導致難以捉摸的錯誤，因為不能保證對象在方法調用发生之前被初始化。

示例：


  abstract class Parent {
    void someMethod() { }
  }

  class Child extends Parent {
    Child() {
      someMethod();
    }

    @Override
    void someMethod() { }
  }

此檢查與以下各項共享其功能：

  • 對象構造期間調用的 abstract 方法檢查
  • 對象構造期間調用的可重寫方法檢查

一次只應啟用一個檢查，以防止重復警告。

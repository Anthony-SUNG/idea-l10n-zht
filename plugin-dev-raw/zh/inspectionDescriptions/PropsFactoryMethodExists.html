報告可將動態構造函數調用安全替換為工廠方法調用的情況。

示例：


  import akka.actor._
  
  class Foo extends AbstractActor
  
  object Foo {
    def props(): Props = Props(new Foo())
  }
  
  Props(classOf[Foo])

在應用快速修復後：


  import akka.actor._
  
  class Foo extends AbstractActor
  
  object Foo {
    def props(): Props = Props(new Foo())
  }
  
  Foo.props()

報告可以遷移到標准 Java API 調用的 Guava 函數基元的用法。

例如，該檢查會報告類和接口（如 FluentIterable、Optional、Function、Predicate 或 Supplier）的用法。

示例：


  ImmutableList<String> results = FluentIterable.from(List.of(1, 2, 3)).transform(Object::toString).toList();

在應用快速修復后：


  List<String> results = List.of(1, 2, 3).stream().map(Object::toString).collect(Collectors.toList());

該快速修復可能會改變語義。 Guava 的一些惰性求值的可迭代對象可以轉換為預求值的可迭代對象。

僅當項目或模塊的語言級別為 8 或更高時，此檢查才會報告。

Reports switch expressions and statements where every branch has common sub-expression, so the switch can be moved inside. 此操作會縮短程式碼。 In many cases, it's reasonable to extract the resulting switch expression to a separate variable or a method.

範例：


  switch (value) {
    case 0 -> System.out.println("zero");
    case 1 -> System.out.println("one");
    case 2, 3, 4 -> System.out.println("few");
    default -> System.out.println("many");
  }

在套用快速修復後：


  System.out.println(switch (value) {
    case 0 -> "zero";
    case 1 -> "one";
    case 2, 3, 4 -> "few";
    default -> "many";
  });

This inspection is applicable only to enhanced switches with arrow syntax.

報告當前文件中未啟用的高級語言功能的用法。

高級語言功能需要通過相應的編譯器選項或通過在使用它們的文件中導入顯式功能標誌來啟用。

後綴運算符表示法：
val s = 123 function toString() { [native code] }

反射調用：
def foo(v: {def bar()}) = v.bar()

動態成員選擇：
class Foo extends Dynamic

隱式轉換：
implicit def toInt(s: String) = s.toInt

高階類型（自 Scala 2.13.1 起默認啟用）：
class Bar[M[A]]

存在類型：
def foo(v: Seq[T] forSome { type T })

宏定義：
def assert(s: String) = macro Asserts.assertImpl

當提供 "-feature" 選項時，Scala 編譯器也可能會發出這些警告。

有關編譯器選項，請參閱：項目設置 | 編譯器 | Scala 編譯器 | 功能

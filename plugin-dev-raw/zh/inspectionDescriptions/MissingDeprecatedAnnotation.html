報告具有 @deprecated Javadoc 標記但沒有 @java.lang.Deprecated 註解的模塊聲明、類、字段或方法。

示例：


  /**
   * @deprecated use {@code example()} instead
   */
  void sample(){ }

在應用快速修復後：


  /**
   * @deprecated use {@code example()} instead
   */
  @Deprecated
  void sample(){ }

此檢查僅在項目或模塊的語言級別為 5 或更高時有效。

使用下面的復選框，對使用 @Deprecated 註解但在 @deprecated Javadoc 標記中沒有解釋的符號發出警告。

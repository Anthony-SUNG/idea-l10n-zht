報告類似于 string(x) 表達式的轉換，其中 x 是整數，但不是字節或 rune。

不建議這種轉換，因為它們返回 Unicode 代碼點 x 的 UTF-8 表示，而不是人們可能預期的返回 x 的十進制字符串表示。 此外，如果 x 表示無效的代碼點，轉換不能被靜態拒絕。

對于打算使用代碼點的轉換，請考慮使用 string(rune(x)) 替換它們。 否則， strconv.Itoa 或其等效函數可以返回基于所需基數值的字符串表示。

示例：

func main() {
 a := 1
 _ = string(a)
}

在應用將整數轉換為 rune快速修復后：

func main() {
 a := 1
 _ = string(rune(a))
}

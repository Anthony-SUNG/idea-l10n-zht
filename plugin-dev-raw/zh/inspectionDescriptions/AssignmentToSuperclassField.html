報告對超類中從子類構造函數中聲明的字段的賦值或修改。

最好在其自己的構造函數中初始化超類的字段，並在子類中委託給該構造函數。 如果它在構造後沒有改變，這還將允許聲明一個字段 final。

示例：


  class Super {
    int x;
  }
  class Sub extends Super {
    Sub(int _x) {
      // Warning: x is declared in a superclass
      x = _x;
    }
  }

要避免此問題，請聲明超類構造函數：


  class Super {
    final int x;

    Super(int _x) {
      x = _x;
    }
  }
  class Sub extends Super {
    Sub(int _x) {
      super(_x);
    }
  }

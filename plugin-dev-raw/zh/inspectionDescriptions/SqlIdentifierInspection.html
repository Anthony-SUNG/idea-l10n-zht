報告在查詢中使用 SQL 預留關鍵字作為標識符名稱時的情況。

示例（Microsoft SQL 服務器）：

CREATE TABLE select (identity INT IDENTITY NOT NULL, order INT NOT NULL);

我們使用 select、identity 和order 作為表名和列名。 但它們也是 Microsoft SQL Server 中的保留關鍵字。 因此，為了將它們用作查詢中的對象名稱，您必須用引號引用這些標識符。 要用引號引用它們，您可以使用為標識符加引號 快速修復。

在應用快速修復後：

CREATE TABLE [select] ([identity] INT IDENTITY NOT NULL, [order] INT NOT NULL);

報告可以替換為數組文字 [...] 的 arrayOf 調用。

示例：

  annotation class MyAnnotation(val strings: Array<String>)

  @MyAnnotation(arrayOf("alpha", "beta", "omega")) // 可替換的 'arrayOf()'
  class MyClass

在應用快速修復後：

  annotation class MyAnnotation(val strings: Array<String>)

  @MyAnnotation(["alpha", "beta", "omega"])
  class MyClass

報告可以取代為字串模板的 function toString() { [native code] } 函數調用。

使用字串模板可使程式碼更簡單。

該快速修復會將 function toString() { [native code] } 取代為字串模板。

示例：


  fun test(): String {
      val x = 1
      return x.function toString() { [native code] }()
  }

在應用快速修復後：


  fun test(): String {
      val x = 1
      return "$x"
  }

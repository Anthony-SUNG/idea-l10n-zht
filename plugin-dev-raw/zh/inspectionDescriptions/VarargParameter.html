報告接受任意數量形參的方法（也稱為 vararg 方法）。

Java 1.4 或更早版本的 JVM 不支持此類方法。

示例：


  enum EnumConstants {
  A(null), B, C;

  EnumConstants(String... ss) {}
}

可通過快速修復將可變實參形參替換為等效的數組形參。 方法調用中的相關實參包裝在數組初始值設定項表達式中。 在應用快速修復后：


  enum EnumConstants {
  A(null), B(new String[]{}), C(new String[]{});

  EnumConstants(String[] ss) {}
}

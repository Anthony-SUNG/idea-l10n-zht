報告對沒有命名實參的數據類的 copy() 方法的調用。

由于 copy() 函數的所有實參都是可選的，因此可能很難理解修改了哪些屬性。 顯式提供形參名稱使得代碼容易理解，且無需導航到 data class 聲明。

示例：


  data class User(val name: String, val age: Int)

  fun copyUser(user: User): User {
      return user.copy("John")
  }

快速修復程序會為所有的 copy() 實參提供形參名稱：


  data class User(val name: String, val age: Int)

  fun copyUser(user: User): User {
      return user.copy(name = "John")
  }

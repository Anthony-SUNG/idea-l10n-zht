Reports JUnit 4 @Before and @After annotated methods and JUnit 5 @BeforeEach and @AfterEach annotated methods that are not public, do not return a void type or take arguments. 此類方法很容易無意間創建，但是 JUnit 測試執行程序不會執行這些方法。

範例：


  @Before private int foo(int arg) { ... }

在套用快速修復後：


  @Before public void foo() { ... }

報告賦值后未使用的賦值。 如果不使用，最好移除該賦值，以縮短代碼並避免冗余分配。

報告的情況如下：

  • 該變量在賦值后從未被讀取。
  • 該變量在被讀取前始終被新值重寫。
  • 變量初始值設定項冗余（出于以上兩種原因之一）。

配置檢查：

使用報告冗余初始值設定項選項可以報告冗余的初始值設定項： 


  int getI() {
    int i = 0; // 冗余初始化
    i = 2;
    return i;
  }

使用在可以替換為(i + 1)時報告 ++i 選項，報告 ++i 表達式可以替換為 i + 1 的情況： 


  int preInc(int i) {
    int res = i;
    return ++res;
  }

使用在變更后的值后續未使用時報告 i++ 選項可以報告后續不使用 i++ 表達式的結果的情況： 


  int postInc(int i) {
    int res = i;
    return res++;
  }

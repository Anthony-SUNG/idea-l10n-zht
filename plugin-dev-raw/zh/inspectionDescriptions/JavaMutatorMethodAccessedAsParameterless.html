報告對具有類似轉變器名稱而沒有實參子句的方法的調用。

Scala 允許在沒有實參子句的情況下調用無參數 Java 方法。 這主要是為了允許 Scala 代碼以與 Scala 訪問器方法相同的方式使用無參數 Java 訪問器方法。 即，不使用任何實參子句並遵守統一訪問原則，也就是說客戶端代碼不應受到將特性實現為字段或方法的決定的影響。

但是，對於具有副作用的方法，通常使用空實參子句。

該快速修復會添加一個空實參子句。

示例：


  val stringBuilder = new java.util.ArrayList[String]
  stringBuilder.clear

在應用快速修復後：


  val stringBuilder = new java.util.ArrayList[String]
  stringBuilder.clear()

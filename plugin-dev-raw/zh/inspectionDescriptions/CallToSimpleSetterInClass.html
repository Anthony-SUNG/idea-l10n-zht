報告從屬性的類內部對簡單屬性 setter 的調用。

簡單的屬性 setter 被定義為只將其形參的值分配給一個字段，而不進行其他計算。 可以安全地內聯此類簡單的 setter 調用。 為了保持代碼清晰，某些編碼標準還建議不使用簡單的 setter。

示例：


  class Foo {
  private int index;
    public Foo(int idx) {
       setIndex(idx);
    }
    public void setIndex(int idx) {
       index = idx;
    }
  }

在應用快速修復後：


  class Foo {
    private int index;
    public Foo(int idx) {
      index = idx;
    }
    public void setIndex(int idx) {
      index = idx;
    }
  }

使用以下選項配置檢查：

  • 是否僅報告 this 上的 setter 調用，而不報告作為形參傳入的相同類型的對象。
  • 是否忽略非 private setter。

報告默認軟件包中的@SpringBootApplication 和冗余的@EnableAutoConfiguration 或 @ComponentScan 注解。

該快速修復會移除冗余注解。

示例：


@SpringBootApplication
@ComponentScan // 報告 '冗余聲明: @SpringBootApplication 已暗示 @ComponentScan'
public class DemoApplication {
    public static void main(String[] args) {
        SpringApplication.run(DemoApplication.class, args);
    }
}

在應用快速修復后：


  @SpringBootApplication
  public class DemoApplication {
    public static void main(String[] args) {
        SpringApplication.run(DemoApplication.class, args);
    }
  }

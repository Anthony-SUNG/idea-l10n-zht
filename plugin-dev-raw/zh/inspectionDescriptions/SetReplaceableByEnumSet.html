報告內容類型為枚舉類的 java.util.Set 對象的實例化。 此類 Set 對象可以取代為 java.util.EnumSet 對象。

與其他集合相比，EnumSet 實現的效率更高，因為底層資料結構是位向量。 使用該快速修復可以將初始值設定項取代為 EnumSet.noneOf() 調用。 當變數類型是 Set 的子類時，此快速修復不可用。

示例：


  enum MyEnum { FOO, BAR; }

  Set<MyEnum> enums = new HashSet<MyEnum>();

在應用快速修復後：


  enum MyEnum { FOO, BAR; }

  Set<MyEnum> enums = EnumSet.noneOf(MyEnum.class);

在使用無參構造函數實例化集合后，立即報告 Collection.addAll() 和 Map.putAll() 調用。

此類結構可以替換為對形參化構造函數的單次調用，從而簡化代碼。 此外，對于某些集合，替換可能會更高效。

示例：

  Set<String> set = new HashSet<>();
  set.addAll(Arrays.asList("alpha", "beta", "gamma"));

可以替換為：

  Set<String> set = new HashSet<>(Arrays.asList("alpha", "beta", "gamma"));

默認情況下支持標准的 JDK 集合類。 此外，還可以在檢查選項中指定其他類。

報告單個 try 語句中相同的 catch 部分。

將這些部分折疊成一個 multi-catch 塊可以減少代碼重復，並防止出現一個 catch 部分更新而另一個部分未更新的情況。

示例：


    try {
        doSmth();
    }
    catch (IOException e) {
        LOG.error(e);
    }
    catch (URISyntaxException e) {
        LOG.error(e);
    }

可通過快速修復使代碼更加緊湊：


    try {
        doSmth();
    }
    catch (IOException | URISyntaxException e) {
        LOG.error(e);
    }

僅當項目或模塊的語言級別為 7 或更高時，此檢查才會報告。

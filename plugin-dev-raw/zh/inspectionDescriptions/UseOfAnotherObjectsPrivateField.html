報告對另一個對象的 private 或 protected 字段的訪問。 Java 允許訪問與當前對象屬于同一類的對象的此類字段，但某些編碼樣式不鼓勵這種用法。 此外，這種對 private 字段的直接訪問在面向組件的架構（例如 Spring 或 Hibernate）中可能會失敗，它們期望對其他對象的所有訪問都通過方法調用進行，以便框架可以使用代理調解訪問。

示例：


  public class Base {
    protected int bar;

    void increment(Base base) {
      bar++;
      base.bar++; // warning: direct access to another object's non-public field
    }
  }
可以通過快速修復封裝該字段。

配置檢查：

  • 使用忽略來自同一類的訪問選項可忽略來自同一類的訪問，只報告來自內部或外部類的訪問。

    要忽略來自內部類的訪問，請使用嵌套的忽略來自內部類的訪問。

  • 使用忽略來自 'equals()' 方法的訪問可忽略來自 equals() 方法的訪問。

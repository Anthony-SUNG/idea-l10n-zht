報告將非安全字符串傳遞給帶有使用注解 org.checkerframework.checker.tainting.qual.Untainted 標記的形參的方法的情況。

安全字符串為：

  • 標記為 @Untainted 的方法調用
  • 未調用非安全方法的局部變量或方法形參
  • 標記為 @Untainted 且沒有分配非安全方法調用的字段、局部變量或形參

示例： 


  void doSmth(boolean b) {
    String s = safe();
    String s1 = "other";
    if (b) s1 = s;
    sink(s);
  }
  
  String sink(@Untainted String s) {}

此處沒有將非安全字符串賦值給 s，因此沒有產生警告。 另一方面： 


  void doSmth(boolean b) {
    String s = safe();
    String s1 = "other";
    s1 = foo();
    if (b) s1 = s;
    sink(s);        // 此處為警告
  }
  
  String foo();

  String sink(@Untainted String s) {}

這裡有一條警告，因為 s1 在 foo 調用結果賦值後具有未知狀態。

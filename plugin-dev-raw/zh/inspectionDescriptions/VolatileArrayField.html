報告被聲明為 volatile 的數組字段。 此類聲明可能令人困惑，因為訪問數組本身遵循 volatile 字段的規則，但訪問數組的內容時並非如此。

示例：


  class Data {
    private volatile int[] idx = new int[0];
  }

如果需要對數組內容進行此類可變訪問，不妨改用 java.util.concurrent.atomic 類： 


  class Data {
    private final AtomicIntegerArray idx = new AtomicIntegerArray(new int[0]);
  }

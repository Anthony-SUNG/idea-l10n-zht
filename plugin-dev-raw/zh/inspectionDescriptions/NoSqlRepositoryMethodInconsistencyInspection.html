報告建置器無法為其生成相應查詢的 Jakarta No SQL 儲存庫方法。

範例：


interface PersonRepository extends Repository<Person, Long> {
  List<Person> findByEmailAndLastname(String email, String lastname);
  List<Person> findByUnknownAndLastname(EmailAddress emailAddress, String lastname); // 無法解析 'Unknown' 屬性
  List<Person> findByAndLastname(String email, String lastname); // 缺少屬性名稱

  List<Person> findByLastnameOrderByFirstnameAsc(String lastname);
  List<Person> findByLastnameOrderByUnknownDesc(String lastname);  // 無法解析 'Unknown' 屬性
}

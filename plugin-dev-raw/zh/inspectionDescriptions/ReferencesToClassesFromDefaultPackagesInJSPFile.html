報告 JSP 文件中對默認軟件包中類的任何引用。

示例：


  <%!
  Integer countSomeUsefulStuff() {
    ...
    new ClassFromDefaultPackage(...); // 錯誤
    new com.example.GoodClass(...); // OK
  }
  %>
  <html>
    <head>
      <title>Hello, JSP!</title>
    </head>
  </html>

注意：在 JSP 編譯期間不會解析此類引用，因為 JSP 的編譯類通常會放在非默認軟件包（例如，Tomcat 的 "org.apache.jsp"，WebSphere 的 "com.ibm._jsp"）中，但在 Java 中不允許從默認軟件包導入。

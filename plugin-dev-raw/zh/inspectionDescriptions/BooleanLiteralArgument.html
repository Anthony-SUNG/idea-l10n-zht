報告類型為 Boolean 但未指定顯式形參名稱的調用實參。

按順序傳遞多個布爾文字時，很容易忘記形參順序，這可能會導致錯誤。 顯式形參名稱讓閱讀和理解代碼變得更容易。

示例：


  fun check(checkName: Boolean, checkAddress: Boolean, checkPhone: Boolean) {}

  fun usage() {
      check(true, false, true) //這是什么意思？
  }

該快速修復會添加缺少的形參名稱：


  fun check(checkName: Boolean, checkAddress: Boolean, checkPhone: Boolean) {}

  fun usage() {
      check(checkName = true, checkAddress = false, checkPhone = true)
  }

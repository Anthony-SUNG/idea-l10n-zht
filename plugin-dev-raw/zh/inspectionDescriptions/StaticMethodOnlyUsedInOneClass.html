報告僅從包含類以外的類使用的 static 方法和欄位。 此類成員可以移動到所使用的類。 此檢查將忽略從成員的類內部的匿名類訪問的工廠方法和成員。 調用同一個類中名稱相同但參數較少的方法的方便重載也會被忽略。

僅在從測試類使用 static 成員時，可以使用第一個復選框來禁止此檢查。

使用下面的第二個復選框可以忽略匿名類、本地類或非 static 內部類中的成員用法。

使用下面的第三個復選框，可以不對沒有問題的情況下也無法移動的成員發出警告，例如，由於目標類中已存在簽名相同的方法，或者由於移動此方法時將無法訪問方法內部使用的欄位或方法。

使用第四個復選框可以忽略位於實用程序類中的成員。

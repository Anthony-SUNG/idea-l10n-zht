報告存儲在實例變量中的未釋放的擁有對象。
如果符合以下條件，您將獲得對象的所有權：
- 使用名稱以 alloc 或 new 開頭或包含 copy
的方法創建 - 向其發送 retain 消息
- 設定具有retain 或 copy 特性並與實例變量合成的屬性的值
要放棄對象的所有權，您需要：
- 向其發送 release 或 autorelease 消息
- 將 nil 分配給與實例變量合成的屬性
如果 ARC（自動引用計數）已啟用，則此檢查會被禁用。 如果下面的復選框被關閉，您可以在類的任何方法中釋放實例變量。 如果打開，您可以在 dealloc、tearDown、applicationWillTerminate:、 didTurnIntoFault 方法以及任何可以從它們調用的方法中釋放實例變量。

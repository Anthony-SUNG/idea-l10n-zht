報告已棄用的 API（類、字段和方法）的用法，例如：new Thread().stop();。

預設情況下，如果不可能或很難避免，則檢查不會發出警告。 比如下面的程式碼就不會被報告：


  abstract class A { //庫程式碼
     @Deprecated
     abstract void m();
  }
  class B extends A { //專案程式碼
     @Override
     void m() {
        //doSmth;
     }
  }

配置檢查：

您可以使用檢查選項在棄用的成員、覆蓋棄用的 abstract 方法、非 static 導入語句、棄用的類的方法或相同的頂級類中禁用此檢查。

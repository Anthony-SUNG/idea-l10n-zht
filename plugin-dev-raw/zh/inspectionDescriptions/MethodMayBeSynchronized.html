報告其主體只包含一條 synchronized 語句的方法。 對于實例方法，此 synchronized 語句的鎖定表達式必須等于 this；對于 static 方法，必須等于 [ClassName].class。

為改善此類方法的可讀性，可以移除 synchronized 包裝器，並將方法標記為 synchronized。

示例：


  public int generateInt(int x) {
    synchronized (this) {
      return 1;
    }
  }

在應用快速修復后：


  public synchronized int generateInt(int x) {
    return 1;
  }

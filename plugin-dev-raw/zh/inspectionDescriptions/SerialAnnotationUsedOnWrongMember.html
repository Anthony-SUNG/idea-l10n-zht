報告 Serializable 和 Externalizable 類中不適合使用 java.io.Serial 注解進行注解的方法和字段。

示例：


class Test implements Serializable {
  @Serial // 注解的字段不是序列化機制的一部分，因為它並非 final
  private static long serialVersionUID = 7874493593505141603L;

  @Serial // 注解的方法不是序列化機制的一部分，因為它並非 private
  void writeObject(ObjectOutputStream out) throws IOException {
  }
}


class Test implements Externalizable {
  @Serial // 該注解方法不是序列化機制的一部分，因為它在 Externalizable 類內部
  private void writeObject(ObjectOutputStream out) throws IOException {
  }
}

有關所有有效情況的信息，請參閱 java.io.Serial 的文檔。

僅當項目或模塊的語言級別為 14 或更高時，此檢查才會報告。

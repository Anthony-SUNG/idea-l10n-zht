報告用作 SLF4J 和 Log4j 2 日志記錄方法實參的非常量字符串串聯。 即使未記錄日志消息，也會在運行時評估非常量串聯；這會對性能產生負面影響。 建議改用形參化日志消息，禁用日志記錄時不會對其進行評估。

示例：


  public class Vital {
    private static final Logger LOG = LoggerFactory.getLogger(Vital.class);

    public void saveTheWorld(int i, String s, boolean b) {
      LOG.info("saveTheWorld(" + i + ", " + s + ", "  + b + ")");
      // todo
    }
  }

在應用快速修復后：


  public class Vital {
    private static final Logger LOG = LoggerFactory.getLogger(Vital.class);

    public void saveTheWorld(int i, String s, boolean b) {
      LOG.info("saveTheWorld({}, {}, {})", i, s, b);
      // todo
    }
  }

配置檢查：

  • 使用警告對象列表可以忽略某些更高的日志記錄級別。 即使在生產中也可以啟用更高的日志記錄級別，並且始終會評估實參。

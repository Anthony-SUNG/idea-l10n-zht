Reports calls to the string methods replace(), replaceAll() or replaceFirst() that have no effect. Such calls can be guaranteed to have no effect when the qualifier and search string are compile-time constants and the search string is not found in the qualifier. 這是冗餘程式碼，可能表示存在編程錯誤。

範例：


  // replacement does nothing
  "hello".replace("$value$", value);

報告不屬于任何類、方法或字段的 Javadoc 注釋。 Javadoc 工具會忽略懸空的 Javadoc 注釋，並且不會將它們包含在其生成的 HTML 文檔中。

示例：


  class A {
    /**
     * Dangling comment
     */
    /**
     * Method javadoc
     */
     public void m(){}
  }

可以通過快速修復完全刪除此類注釋或將它們轉換為塊注釋。 在應用快速修復后：


  class A {
    /*
      Dangling comment
    */
    /**
     * Method javadoc
     */
    public void m(){}
  }

使用忽略 JavaDoc 格式的文件頭注釋選項可以忽略 Java 文件開頭的注釋。 這些通常是版權消息。

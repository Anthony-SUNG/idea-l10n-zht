報告未被聲明為 public 和 static、不返回 void 或接受實參的 JUnit 4.0 @BeforeClass 或 @AfterClass 方法。 此類方法很容易無意間創建，但是 JUnit 測試運行程序不會執行這些方法。

示例：


  public class MainTest {
    @BeforeClass
    String beforeClass(int i) {
    }
  }

應用快速修復后，該方法變為：


  class MainTest {
    @BeforeClass
    public static void beforeClass() {
    }
  }

報告從不符合所聲明方法返回范圍的方法返回的數值。 您可以使用許多注解聲明方法的返回范圍：
  • JetBrains 注解軟件包中的 org.jetbrains.annotations.Range（指定 'from' 和 'to'）
  • Checker Framework 注解軟件包中的 org.checkerframework.common.value.qual.IntRange（指定 'from' 和 'to'）
  • Checker Framework 注解軟件包中的 org.checkerframework.checker.index.qual.GTENegativeOne（范圍為 '>= -1'）
  • Checker Framework 注解軟件包中的 org.checkerframework.checker.index.qual.NonNegative（范圍為 '>= 0'）
  • Checker Framework 注解軟件包中的 org.checkerframework.checker.index.qual.Positive（范圍為 '> 0'）
  • JSR 305 注解軟件包中的 javax.annotation.Nonnegative（范圍為 '>= 0'）
  • javax.validation.constraints.Min（指定最小值）
  • javax.validation.constraints.Max（指定最大值）

示例：


  @Range(from = 0, to = Integer.MAX_VALUE) int getValue() {
    // 警告：-1 超出聲明的范圍
    return -1;
  }

報告對其中的調用由聲明類的子類而不是聲明類本身限定的內部類和嵌套類的訪問。

Java 允許此類限定，但是此類訪問可能表明繼承和重寫之間存在難以捉摸的混淆。

示例：


  class Super {
    static class Inner {}
  }

  class Sub extends Super {
    void test() {
      Sub.Inner s = new Sub.Inner(); // 'Inner' 類在 'Super' 類中聲明，但通過 'Sub' 類引用
    }
  }

在應用快速修復后：


  class Super {
    static class Inner {}
  }

  class Sub extends Super {
    void test() {
      Super.Inner s = new Super.Inner();
    }
  }

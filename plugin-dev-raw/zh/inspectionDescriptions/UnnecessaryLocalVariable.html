報告不必要的、不會增加方法的可理解性的局部變量，包括：
  • 立即返回的局部變量。
  • 立刻賦值給另一個變量並且不再使用的局部變量。
  • 始終與另一個局部變量或形參具有相同值的局部變量。

示例：


  boolean yes() {
    boolean b = true;
    return b;
  }

在應用快速修復後：


  boolean yes() {
    return true;
  }

配置檢查：

  • 使用忽略立即返回或拋出的變量選項可以忽略立即返回或拋出的變量。 為清晰起見和易於調試，某些編碼樣式建議使用此類變量。
  • 使用忽略具有註解的變量選項可以忽略註解的變量。

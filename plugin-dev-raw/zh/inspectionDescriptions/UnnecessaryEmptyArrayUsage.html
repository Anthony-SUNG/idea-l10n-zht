報告數組元素類型的類中有常量時已知長度為零的數組分配。 由于零長度數組不可變，您可以重用相同的數組實例來節省內存。

示例：


  class Item {
    // 可重用的 public 零長度數組常量 
    public static final Item[] EMPTY_ARRAY = new Item[0];
  }
  class EmptyNode {
    Item[] getChildren() {
      // 不必要的零長度數組創建
      return new Item[0];
    }
  }

在應用快速修復后：


  class EmptyNode {
    Item[] getChildren() {
      return Item.EMPTY_ARRAY;
    }
  }

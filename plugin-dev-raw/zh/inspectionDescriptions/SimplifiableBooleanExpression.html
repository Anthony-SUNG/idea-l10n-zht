報告可以簡化的布爾表達式。

示例：


  void f(boolean foo, boolean bar) {
    boolean b = !(foo ^ bar);
  }

在應用快速修復后：


  void f(boolean foo, boolean bar) {
    boolean b = foo == bar;
  }

示例：


  void f(boolean foo, boolean bar) {
    boolean b = (foo && bar) || !foo;
  }

在應用快速修復后：


  void f(boolean foo, boolean bar) {
    boolean b = !foo || bar;
  }

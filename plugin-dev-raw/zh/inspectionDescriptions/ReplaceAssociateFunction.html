報告對可以替換為 associateBy() 或 associateWith() 的 associate() 和 associateTo() 的調用。

兩個函數都接受應用于給定序列或集合（作為接收器）的元素的轉換器函數。 然后該對函數被用于構建結果 Map。

鑑于轉換器引用 it，associate[To]() 調用可以替換為性能更高的 associateBy() 或 associateWith()。

示例：

  fun getKey(i: Int) = 1L
  fun getValue(i: Int) = 1L

  fun test() {
      arrayOf(1).associate { getKey(it) to it }  // 可替換的 'associate()'
      listOf(1).associate { it to getValue(it) } // 可替換的 'associate()'
  }

在應用快速修復后：

  fun getKey(i: Int) = 1L
  fun getValue(i: Int) = 1L

  fun test() {
      arrayOf(1).associateBy { getKey(it) }
      listOf(1).associateWith { getValue(it) }
  }

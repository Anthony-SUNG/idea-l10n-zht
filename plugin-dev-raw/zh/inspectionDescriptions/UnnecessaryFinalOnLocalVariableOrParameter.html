報告不必要地聲明為 final 的局部變量或形參。

出於簡潔的原因，某些編碼標準不贊成聲明為 final 的變量。

示例：


  class Foo {
    Foo(Object o) {}

    void bar(final Object o) {
      new Foo(o);
    }
  }

在應用快速修復後：


  class Foo {
    Foo(Object o) {}

    void bar(Object o) {
      new Foo(o);
    }
  }

使用檢查選項可以切換以下各項的報告：

  • 局部變量
  • 形參（包括 catch 塊的形參和增強型 for 語句）

此外，您可以將檢查配置為僅報告 abstract 或接口方法的 final 形參，這可能被認為是多餘的，因為此類標記不會影響這些方法的實現。

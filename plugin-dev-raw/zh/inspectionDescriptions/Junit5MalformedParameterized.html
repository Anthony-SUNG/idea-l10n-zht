報告源格式錯誤的形參化測試：

  • MethodSource 具有未知目標或方法並非 static、no-arg。
  • ValueSource 和 EnumSource 類型不能轉換為方法形參。
  • 未定義來源。

示例：


  class Test {
    @MethodSource("parameters")
    @ParameterizedTest
    void foo(String param) {}
  }

在應用快速修復后：


  class Test {
    private static Stream<Arguments> parameters() {
        return Stream.empty();
    }

    @MethodSource("parameters")
    @ParameterizedTest
    void foo(String param) {}
  }

報告來源格式錯誤的參數化測試：
  • MethodSource 具有未知目標或方法不為 static、無實參。
  • ValueSource / EnumSource 類型不能轉換為方法參數。
  • 未定義來源。

示例：


  class Test {
    @MethodSource("parameters")
    @ParameterizedTest
    void foo(String param) {}
  }

在應用快速修復後：


  class Test {
    private static Stream<Arguments> parameters() {
        return Stream.empty();
    }

    @MethodSource("parameters")
    @ParameterizedTest
    void foo(String param) {}
  }

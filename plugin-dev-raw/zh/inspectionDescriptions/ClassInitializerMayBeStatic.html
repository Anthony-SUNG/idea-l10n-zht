報告可以使之為 static 的實例初始值設定項。

如果實例初始值設定項不引用其類的任何非 static 成員，則可以為 static。 static 初始值設定項在類解析後執行，而實例初始值設定項對此類的每個實例化執行。

範例：


  class A {
     public static String CONSTANT;
     {
        CONSTANT = "Hello";
     }
  }

在應用快速修復後：


  class A {
     public static String CONSTANT;
     static {
        CONSTANT = "Hello"; // 現在每個類僅實例化一次
     }
  }

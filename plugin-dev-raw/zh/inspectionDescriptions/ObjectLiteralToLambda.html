報告使用單個 abstract 方法實現 Java 接口的匿名對象文字，該方法可使用 lambda 表達式轉換為調用。

示例：


class SomeService {
  val threadPool = Executors.newCachedThreadPool()
    
  fun foo() {
    threadPool.submit(object : Runnable {
      override fun run() {
        println("hello")
      }
    })
  }
}

在應用快速修復後：


  fun foo() {
    threadPool.submit { println("hello") }
  }

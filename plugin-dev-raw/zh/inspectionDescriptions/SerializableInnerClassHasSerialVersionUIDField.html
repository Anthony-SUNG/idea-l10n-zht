報告實現 java.io.Serializable 但不定義 serialVersionUID 字段的非 static 內部類。

如果沒有 serialVersionUID 字段，對該類做出的任何更改都將導致以前序列化的版本不可讀。 強烈建議 Serializable 非 static 內部類包含 serialVersionUID 字段，否則由於合成訪問器方法的差異，默認序列化算法可能導致編譯器之間的序列化版本不兼容。

建議通過快速修復添加缺失的 serialVersionUID 字段。

示例：


  class Outer {
      class Inner implements Serializable {}
  }

在應用快速修復後：


  class Outer {
      class Inner implements Serializable {
          private static final long serialVersionUID = -7004458730436243902L;
      }
  }

使用以下選項配置檢查：

  • 列出此檢查不應報告其繼承者的類。 這適用於從超類繼承 Serializable 但不用於序列化的類。
  • 是否忽略 Serializable 匿名類。

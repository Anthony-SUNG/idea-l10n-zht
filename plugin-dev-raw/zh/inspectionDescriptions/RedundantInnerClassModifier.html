如果類沒有引用其外部類的成員，則將類上的 inner 修飾符報告為冗余。

示例：


  class Foo {
      inner class InnerClass { // 冗余 `inner` 修飾符
          fun hello() {
              println("Hi!")
          }
      }
  }

  class List {
      val objects = Array<Any>(42) { Any() }

      inner class Iterator { // 非冗余 `inner` 修飾符
          fun next(): Any {
              return objects[0]
          }
      }
  }

在應用快速修復后：


  class Foo {
      class InnerClass { // 冗余 `inner` 修飾符
          fun hello() {
              println("Hi!")
          }
      }
  }

  class List {
      val objects = Array<Any>(42) { Any() }

      inner class Iterator { // 非冗余 `inner` 修飾符
          fun next(): Any {
              return objects[0]
          }
      }
  }

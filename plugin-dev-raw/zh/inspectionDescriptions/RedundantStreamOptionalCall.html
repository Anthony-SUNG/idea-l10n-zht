Reports redundant Stream or Optional calls like map(x -> x), filter(x -> true) or redundant sorted() or distinct() calls.

Note that a mapping operation in code like streamOfIntegers.map(Integer::valueOf) works as requireNonNull() check: if the stream contains null, it throws a NullPointerException, thus it's not absolutely redundant. Disable the Report redundant boxing in Stream.map() option if you do not want such cases to be reported.

僅當專案或模組的語言級別為 8 或更高時，此檢查才會報告。

報告實現兩個或多個特征（包含具有相同簽名的方法）的類。

調用此類方法可能產生意外的結果。

該快速修復會添加顯式重寫方法。

示例：


  trait T1 {
      def foo() {}
  }

  trait T2 {
      def foo() {}
  }

  class X implements T1, T2 {}

  // 將調用 T2.foo()
  new X().foo()

在應用快速修復后：


  class X implements T1, T2 {
      @Override
      Object foo() {
          return T2.super.foo()
      }
  }

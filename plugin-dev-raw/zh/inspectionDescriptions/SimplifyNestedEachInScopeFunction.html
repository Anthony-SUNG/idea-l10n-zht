報告 also 或 apply 等範圍函數中可以簡化的 forEach 函數。

將 forEach 調用轉換為 onEach 快速修復可用於自動修正代碼。

示例：


  fun test(list: List<Int>) {
      val x = list.also { it.forEach { it + 4 } }.function toString() { [native code] }()
      val y = list.apply { forEach { println(it) } }
  }

在應用快速修復後：


  fun test(list: List<Int>) {
      val x = list.onEach { it + 4 }.function toString() { [native code] }()
      val y = list.onEach { println(it) }
  }

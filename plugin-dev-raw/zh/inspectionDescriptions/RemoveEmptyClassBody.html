報告具有空主體的類和對象的聲明。

使用“移除冗余的空類主體”快速修復可清理代碼。

示例：


  class EmptyA() {} // <== 空主體

  class EmptyB {
      companion object {} // <== 空主體
  }

  fun emptyC() {
     object {} // <== 匿名對象，可以（未報告）
  }

在應用快速修復后：


  class EmptyA()

  class EmptyB {
      companion object
  }

  fun emptyC() {
     object {}
  }

報告可替換為 to() 中綴函數調用的 Pair 構造函數調用。

顯式構造函數調用可能會增添冗余，尤其是在多次使用時。 將構造函數調用替換為 to() 會使讀取和維護代碼變得更容易。

示例：


  val countries = mapOf(
      Pair("France", "Paris"),
      Pair("Spain", "Madrid"),
      Pair("Germany", "Berlin")
  )

在應用快速修復后：


  val countries = mapOf(
      "France" to "Paris",
      "Spain" to "Madrid",
      "Germany" to "Berlin"
  )

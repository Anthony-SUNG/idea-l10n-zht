報告冗餘的 String 構造函數和對可以替換為簡化表達式的 function toString() { [native code] }() 或 substring() 等方法的調用。

例如，在 "string".substring(0)、"string".function toString() { [native code] }() 或 new StringBuilder().function toString() { [native code] }().substring(1,3) 等情況下，可以安全移除對這些方法的調用。

示例：


  System.out.println(new String("message"));

在應用快速修復後：


  System.out.println("message");

請注意，該快速修復會移除冗餘構造函數，這可能會影響 String 引用相等。 即使被認為是一種不好的做法，如果您仍需要保留，請禁止警告或將設置配置為忽略冗餘的 String 構造函數。

報告用于將 Web 模塊部署到服務器的描述符中的以下問題：
  • 對不可實例化類的引用
  • 對不擴展所需類的類的引用
  • 對范圍不適當的類的引用
  • 空標記和特性值
  • 與所需模式不匹配的標記和特性值（例如，Java 標識符）
  • 不包含必需子標記或特性的標記
  • 定義具有重復名稱的對象的標記

示例：



<web-app>
  <filter-mapping>
    <filter-name>CustomFilter</filter-name> // 錯誤：應定義 servlet-name 或 url-pattern 子標記
  </filter-mapping>
  <servlet-mapping>
    <servlet-name>CustomServletName</servlet-name>
    <url-pattern>/ baseUrl</url-pattern> // 錯誤：URI 模式不得包含空格
  </servlet-mapping>
</web-app>

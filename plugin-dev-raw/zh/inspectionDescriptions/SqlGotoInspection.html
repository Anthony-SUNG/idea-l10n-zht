報告後向 GOTO 語句和用於退出循環的 GOTO 語句的用法。

通常不推薦大量使用 GOTO 語句。 有關詳細資訊，請參閱 ibm.com 上的 GOTO statement in SQL procedures 頁面。

考慮使用循環而不是使用 GOTO 返回到上一條語句。

考慮使用另一個流控制語句，例如 RETURN 或 BREAK，而不是使用 GOTO 退出 WHILE 循環。

示例（Oracle）：

CREATE PROCEDURE test(n INT) AS
DECLARE
    x INT;
BEGIN
    x := 0;
    GOTO a;
    <<a>> x := 1;
    IF (n = 0) THEN
        GOTO a;
    END IF;
    WHILE TRUE
        LOOP
            GOTO b;
        END LOOP;
    <<b>> x := 3;
END;

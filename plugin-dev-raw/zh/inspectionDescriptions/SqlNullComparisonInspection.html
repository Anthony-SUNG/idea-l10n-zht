報告與可以替換為 IS NULL 或 IS NOT NULL 運算符的 NULL 的比較。

示例（Microsoft SQL 伺服器）：

CREATE TABLE foo ( id int );

SELECT * FROM foo WHERE NULL = NULL;
SELECT * FROM foo WHERE NULL != NULL;

NULL = NULL 可以替換為 IS NULL，NULL != NULL 比較可以替換為 IS NOT NULL。 要進行此替換，您可以使用使用 IS NULL 運算符或使用 IS NOT NULL 運算符快速修復。

SELECT * FROM foo WHERE NULL IS NULL;
SELECT * FROM foo WHERE NULL IS NOT NULL;

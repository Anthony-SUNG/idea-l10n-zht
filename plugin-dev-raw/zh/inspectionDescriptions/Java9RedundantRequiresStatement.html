報告 Java Platform Module System module-info.java 檔案中的冗餘 requires 指令。 當模塊 A 需要模塊 B，但模塊 A 中的程式碼不從 B 匯入任何軟件包或類時，則 requires 指令冗餘。 此外，所有模塊都隱式地宣告了對 java.base 模塊的依賴，因此 requires java.base; 指令始終冗餘。

快速修復會刪除冗餘的 requires 指令。 如果刪除的依賴項重新匯出了實際使用的模塊，則修復程序會為這些模塊添加 requires 指令。

僅當專案或模塊的語言級別為 9 或更高時，此檢查才會報告。

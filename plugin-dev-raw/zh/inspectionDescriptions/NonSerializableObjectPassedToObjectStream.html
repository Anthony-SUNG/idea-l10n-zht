報告用作 java.io.ObjectOutputStream.write() 實參的非 Serializable 對象。 此類調用將導致執行時異常。

此檢查假定 java.util.Collection 和 java.util.Map 類型的對象為 Serializable，除非聲明它們的類型為非 Serializable。

示例：


  public class IWantToSerializeThis {
    public static void main(String[] args) throws IOException {
      try(var stream = new ObjectOutputStream(Files.newOutputStream(Paths.get("output")))) {
        // 警告 -- 將會失敗並拋出 NotSerializableException
        stream.writeObject(new IWantToSerializeThis());
      }
    }
  }

報告可以取代為方法引用的匿名類。

請注意，如果將匿名類轉換為未綁定的方法引用，則 Java 運行時可以在後續調用期間重用相同的方法引用對象。 另一方面，使用匿名類時，每次都會創建單獨的對象。 因此，在極少數情況下，應用該快速修復可能會導致語義發生變化，例如，當匿名類實例用作 HashMap 鍵時。

示例：


    Runnable r = new Runnable() {
      @Override
      public void run() {
        System.out.println();
      }
    };

該快速修復會將此代碼更改為壓縮形式：Runnable r = System.out::println;。

使用在接口未使用 @FunctionalInterface 註解時報告選項可為未使用 @FunctionalInterface 註解的接口啟用此檢查。

僅當項目或模塊的語言級別為 8 或更高時，此檢查才會報告。

報告調用 ReactorDebugAgent.init() 時 Hooks.onOperatorDebug() 的用法。

如果 ReactorDebugAgent 在類路徑中，並且 spring.reactor.debug-agent.enabled 為 true（默認值），啟動時自動調用 ReactorDebugAgent.init()。 同時使用 Hooks.onOperatorDebug() 和 ReactorDebugAgent.init() 會導致 Reactor 生成雙倍的調試堆棧幀。 此外，Hooks.onOperatorDebug() 會導致性能開銷。

該快速修復會移除對 Hooks.onOperatorDebug() 的調用。

示例：


  public void hook() {
    Hooks.onOperatorDebug(); // 報告 '初始化 ReactorDebugAgent 時調用 Hooks.onOperatorDebug()'
  }

在應用快速修復后：


  public void hook() {
  }

此檢查僅在項目或模塊的 Spring Boot 版本為 2.2.0 或更高時觸发。

報告可以比較兩個數組的 equals() 調用。

對數組調用 equals() 可以比較同一性，相當於使用 ==。 使用 Arrays.equals() 比較兩個數組的內容，或使用 Arrays.deepEquals() 比較多維數組的內容。

示例：


  void sample(int[] first, int[] second){
    if (first.equals(second)) return;
  }

在應用快速修復後：


  void sample(int[] first, int[] second){
    if (Arrays.equals(first, second)) return;
  }

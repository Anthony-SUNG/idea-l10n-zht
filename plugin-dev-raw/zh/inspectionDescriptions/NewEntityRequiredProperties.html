報告使用 new 或 getOrCreate 函數在 ChangeScope 中創建時 Entity 的未分配屬性。

必須分配 Entity 的所有非可 null 屬性，或者使用 @Many 注解對其進行標記。 否則，可能會導致運行時異常。

示例：


  interface A: Entity {
    var x: Int
    var y: Int?
    @Many var z: Int
    var p: Int
  }

  fun ChangeScope.foo() {
    new(A::class) {
      this.p = 0
    }
  }

快速修復會生成缺少必需屬性的賦值


  fun ChangeScope.foo() {
    new(A::class) {
      this.x = x
      this.p = 0
    }
  }

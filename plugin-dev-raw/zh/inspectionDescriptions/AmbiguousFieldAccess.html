如果在類周圍的程式碼中存在同名的局部變數、參數或欄位，則報告從匿名類、內部類或局部類對超類欄位的訪問。 在這種情況下，看似訪問了來自周圍程式碼的元素，而實際上訪問的是來自超類的欄位。

為闡明程式碼的意圖，建議對欄位訪問添加顯式 super 限定符。

範例：


    class First {
      protected String ambiguous;
    }
    class Second {
      void foo(String ambiguous) {
        new First() {
          {
            System.out.println(ambiguous); // 訪問的是欄位，而不是參數
          }
        };
      }
    }

在應用快速修復後：


    class First {
      protected String ambiguous;
    }
    class Second {
      void foo(String ambiguous) {
        new First() {
          {
            System.out.println(super.ambiguous);
          }
        };
      }
    }

如果在類周圍的代碼中存在同名的局部變量、參數或字段，則報告從匿名類、內部類或局部類對超類字段的訪問。 在這種情況下，看似訪問了來自周圍代碼的元素，而實際上訪問的是來自超類的字段。

為闡明代碼的意圖，建議對字段訪問添加顯式 super 限定符。

示例：


    class First {
      protected String ambiguous;
    }
    class Second {
      void foo(String ambiguous) {
        new First() {
          {
            System.out.println(ambiguous); // 訪問的是字段，而不是參數
          }
        };
      }
    }

在應用快速修復後：


    class First {
      protected String ambiguous;
    }
    class Second {
      void foo(String ambiguous) {
        new First() {
          {
            System.out.println(super.ambiguous);
          }
        };
      }
    }

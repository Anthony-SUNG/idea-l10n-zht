報告實現 java.io.Serializable 且其中的 readResolve() 或 writeReplace() 方法未被聲明為 protected 的類。

將 readResolve() 和 writeReplace() 方法聲明為 private 可以強制子類無提示地忽略它們，而聲明為 public 則允許不受信任的代碼調用它們。

如果包含類被聲明為 final，這些方法就可以被聲明為 private。

示例：


  class ClassWithSerialization implements Serializable {
    public Object writeReplace() { // 警告：'writeReplace()' 未被聲明為 protected
        ...
    }
  }

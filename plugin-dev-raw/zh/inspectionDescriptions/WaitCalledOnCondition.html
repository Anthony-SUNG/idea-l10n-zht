報告在 java.util.concurrent.locks.Condition 對象上進行的 wait() 調用。 這可能是編程錯誤，本應使用 await() 方法的某種變體。

示例：


  void acquire(Condition released) throws InterruptedException {
    while (acquired) {
      released.wait();
    }
  }

優良的代碼應類似于：


  void acquire(Condition released) throws InterruptedException {
    while (acquired) {
      released.await();
    }
  }

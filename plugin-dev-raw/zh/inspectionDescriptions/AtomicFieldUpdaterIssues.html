報告 AtomicLongFieldUpdater、AtomicIntegerFieldUpdater 或 AtomicReferenceFieldUpdater 字段（java.util.concurrent.atomic 軟件包）的問題。

報告的問題與原子字段更新程序可能發生的運行時問題相同：找不到指定字段、指定字段不可訪問、指定字段類型錯誤等。

示例：

  • 
  class A {
    private static volatile int value = 0;
    private static final AtomicIntegerFieldUpdater updater =
    AtomicIntegerFieldUpdater.newUpdater((A.class), "value"); // 警告：字段 'value' 具有 'static' 修飾符
  }
  
  • 
  class B {
    private static final AtomicIntegerFieldUpdater updater =
      AtomicIntegerFieldUpdater.newUpdater(B.class, "value"); // 警告：在類 'B' 中找不到名為 'Value' 的字段
  }

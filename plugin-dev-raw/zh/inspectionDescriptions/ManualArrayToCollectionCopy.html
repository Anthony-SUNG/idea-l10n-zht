報告使用循環將數組內容復制到集合中的代碼。

一種更短並且有可能更快（取決于集合實現）的方法是使用 Collection.addAll(Arrays.asList()) 或 Collections.addAll()。

僅報告內部沒有附加語句的循環。

示例：


  void addAll(List<String> list, String[] arr) {
    for (int i = 0; i < arr.length; i++) {
      String s = arr[i];
      list.add(s);
    }
  }

在應用快速修復后：


  void addAll(List<String> list, String[] arr) {
    Collections.addAll(list, arr);
  }

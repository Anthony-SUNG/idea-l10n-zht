報告重寫 abstract 成員但沒有 override 修飾符的函數。

Scala 不要求開发者對實現 abstract 成員的方法使用 override。 盡管如此，最好用 override 標記此類方法，以便在重寫關系被損壞時（例如，通過后續重構）由編譯器通知。

示例：


  trait Base {
    def fun(): Int
  }

  class Impl extends Base {
    def fun(): Int = 42
  }

在應用快速修復后：


  trait Base {
    def fun(): Int
  }

  class Impl extends Base {
    override def fun(): Int = 42
  }

報告重寫 abstract 成員但沒有 override 修飾符的函數。

Scala 不要求開發者對實現 abstract 成員的方法使用 override。 儘管如此，最好用 override 標記此類方法，以便在重寫關係被損壞時（例如，通過後續重構）由編譯器通知。

示例：


  trait Base {
    def fun(): Int
  }

  class Impl extends Base {
    def fun(): Int = 42
  }

在應用快速修復後：


  trait Base {
    def fun(): Int
  }

  class Impl extends Base {
    override def fun(): Int = 42
  }

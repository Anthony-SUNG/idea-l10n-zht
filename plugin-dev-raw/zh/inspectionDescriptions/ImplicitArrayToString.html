報告 String 連接中使用的數組，或作為參數傳遞給 java.io.PrintStream 方法的數組，如 System.out.println()。

通常使用數組的內容，而不是使用數組對象本身。

示例：


  void print(Object[] objects) {
    System.out.println(objects);
  }

在應用快速修復後：


  void print(Object[] objects) {
    System.out.println(Arrays.function toString() { [native code] }(objects));
  }

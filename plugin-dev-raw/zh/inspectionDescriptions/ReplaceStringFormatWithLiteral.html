報告可以替換為字符串模板的 String.format 調用。

使用字符串模板可使代碼更簡單。

該快速修復會將調用替換為字符串模板。

示例：


  fun main() {
      val id = "abc"
      val date = "123"
      val s = String.format("%s_%s_%s", id, date, id)
  }

在應用快速修復後：


  fun main() {
      val id = "abc"
      val date = "123"
      val s = "${id}_${date}_$id"
  }

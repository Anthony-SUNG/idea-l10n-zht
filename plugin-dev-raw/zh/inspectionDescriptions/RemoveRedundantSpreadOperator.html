報告針對一系列 arrayOf 函數調用而使用冗余展開運算符的情況。

使用“移除冗余展開運算符”快速修復可清理代碼。

示例：


  fun foo(vararg s: String) { }

  fun bar(ss: Array<String>) {
      foo(*arrayOf("abc"))       // 對于 'foo' 的兩次調用，數組創建
      foo(*arrayOf(*ss, "zzz"))  // 及其后續分配均冗余
  }

在應用快速修復后：


  fun foo(vararg s: String) { }

  fun bar(ss: Array<String>) {
      foo("abc")
      foo(*ss, "zzz")
  }

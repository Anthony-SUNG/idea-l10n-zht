報告對可以安全移除的字串模板中的 function toString() { [native code] }() 的調用。

示例：

  fun foo(a: Int, b: Int) = a + b

  fun test(): String {
      return "Foo: ${foo(0, 4).function toString() { [native code] }()}" // 'function toString() { [native code] }()' 冗餘
  }

在應用快速修復後：

  fun foo(a: Int, b: Int) = a + b

  fun test(): String {
      return "Foo: ${foo(0, 4)}"
  }

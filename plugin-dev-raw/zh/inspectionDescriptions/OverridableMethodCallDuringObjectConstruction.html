報告在對象構造期間對當前類的可重寫方法的調用。

如果方法位于以下位置，則會在對象構造期間調用：

  • 構造函數
  • 非 static 實例初始值設定項
  • 非 static 字段初始值設定項
  • clone() 方法
  • readObject() 方法
  • readObjectNoData() 方法

    • 如果方法未被聲明為 final、static 或 private，則該方法可重寫。 即使可重寫，package-local 方法也被認為是安全的。 此類調用可能會導致難以捉摸的錯誤，因為對象初始化可能发生在方法調用之前。

    示例：

    
  class Parent {
    void someMethod() { }
  }

  class Child extends Parent {
    Child() {
      someMethod();
    }
  }

    此檢查與以下檢查共享功能：

    • 對象構造期間調用的 abstract 方法
    • 在對象構造期間調用的重寫的方法

    一次只能啟用一個檢查，以防止警告重復。

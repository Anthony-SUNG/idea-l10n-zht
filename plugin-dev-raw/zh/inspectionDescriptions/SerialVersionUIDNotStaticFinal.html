報告 serialVersionUID 字段未被聲明為 private static final long 的 Serializable 類。

示例：


  class SampleClass implements Serializable {
    private long serialVersionUID = 1; // Serializable 類的字段未聲明為 'private static final long'

    public SampleClass() {
        System.out.println(serialVersionUID);
    }
  }

適用于通過在項目工具窗口中調用新建 | Java 類 | 注解創建的新 Java 注解。

此內置模板是可編輯的。 除了 Java 表達式和注釋外，您還可以使用預定義變量（如下所列），這些變量隨后將像宏一樣被擴展為相應值。

您也可以指定自定義變量。 自定義變量使用以下格式：${VARIABLE_NAME}，其中 VARIABLE_NAME是變量的名稱（例如，${MY_CUSTOM_FUNCTION_NAME}） 。 在 IDE 創建包含自定義變量的新文件之前，您會看到一個對話框，您可以在其中定義模板中自定義變量的值。

通過使用 #parse 指令，可以包括 包含 選項卡中的模板。 要包含模板，請在引號中將模板的全名指定為形參（例如，#parse("File Header.java")）。
預定義變量接收以下值：
${PACKAGE_NAME}   在其中創建新注解的軟件包名稱
${NAME}   您在創建新類對話框中指定的新注解的名稱
${USER}   當前用戶的系統登錄名
${DATE}   當前系統日期
${TIME}   當前系統時間
${YEAR}   當前年度
${MONTH}   當前月份
${MONTH_NAME_SHORT}   當前月份名稱的前 3 個字母（例如，Jan、Feb 等）
${MONTH_NAME_FULL}   當前月份的全名（January、February 等）。
${DAY}   當前月份的日期
${HOUR}   當前小時
${MINUTE}   當前分鐘
${PROJECT_NAME}   當前項目名稱

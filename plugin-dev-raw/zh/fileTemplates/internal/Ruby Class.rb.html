一個 Ruby 類的內置模板。 每次您創建新的 Ruby 類時都會使用此模版。 從項目視圖之一的彈出菜單中選擇 新建 | Ruby 文件/類 。 除了 Ruby 語句、表達式和注釋外，您還可以使用下面列出的預定義變量。 每個變量都像宏一樣擴展為相應的值。 通過 #parse 指令，可以通過將所需模板的全名指定為引號中的形參，從而包含 Includes 選項卡中的模板。
預定義變量接收以下值：
${FILE_NAME}   當前文件名稱
${USER}   當前的用戶系統登錄名
${DATE}   當前系統日期
${TIME}   當前系統時間
${YEAR}   當前年度
${MONTH}   當前月份
${DAY}   當前月份的日期
${HOUR}   當前小時
${MINUTE}   當前分鐘
${PRODUCT_NAME}   當前 IDE 名稱
${PROJECT_NAME}   當前項目名稱

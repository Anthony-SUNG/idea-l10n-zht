在生成已實現下標（例如，使用代碼 | 生成 | 實現方法操作）時填充下標的主體。
模板是可編輯的。 除了 Swift 表達式和注釋外，您還可以使用預定義變量，這些變量隨后將被擴展為相應值。
預定義變量接收以下值：
${RETURN_TYPE}   創建的下標的返回類型
${DEFAULT_RETURN_VALUE}   返回值的建議默認值（可以為空）
${SIGNATURE}   下標簽名
${HAS_SET}   當下標有 setter 時，為 "true"
${GETTER_EFFECTS}   屬性 getter 的 getter 效果（async、throws）
${GET_ATTRIBUTES}   下標 getter 的特性
${SET_ATTRIBUTES}   下標 setter 的特性
Swift 文件頭模板（包含選項卡）中的所有預定義變量也可用（例如 ${USER}、${DATE} 等）。

模板是可編輯的。 除了 Objective-C 表達式和注釋外，您還可以使用預定義變量，這些變量隨后將被擴展為相應值。
預定義變量將接收以下值：
${CALL_SUPER}   super 方法調用，對于 void 方法 — [super methodName]，帶或不帶形參； 對于其他方法 — return [super methodName]，帶或不帶形參
${RETURN_TYPE}   創建的方法的返回類型
${DEFAULT_RETURN_VALUE}   方法默認返回的值
${CONTAINING_CLASS}   在其中重寫方法的類的名稱
${CUSTOM_CODE}   type-specific isEqualToType：比較方法名稱（如果已生成）
C 文件標頭模板（包含選項卡）中的所有預定義變量也可用（${USER}、${DATE} 等）

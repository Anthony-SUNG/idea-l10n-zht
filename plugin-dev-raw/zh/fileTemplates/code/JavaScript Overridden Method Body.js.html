此內置模板用於在每次生成被重寫方法（例如，調用代碼 | 重寫方法… 功能）時填寫方法體。
模板是可編輯的。 除了 Javascript 表達式和注釋，您還可以使用預定義變量。 當模板被應用時，這些變量將被替換成相應的值。
預定義變量接收以下值：
${CALL_SUPER}   super 方法調用，對於 void 方法 - super.method_name(); 帶或不帶形參； 對於其他方法 - return super.method_name(); 帶或不帶形參
${PLAIN_CALL_SUPER}   super 方法調用，super.method_name(); 帶或不帶形參；
${RETURN_TYPE}   創建方法的返回類型
${DEFAULT_RETURN_VALUE}   方法默認返回的值
${METHOD_NAME}   被重寫的方法的名稱
所有來自 文件頭 模板（包含選項卡）的預定義變量也可用。 （例如： ${USER}、 ${DATE} 等。）

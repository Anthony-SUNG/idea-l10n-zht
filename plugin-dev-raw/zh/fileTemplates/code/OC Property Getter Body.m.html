生成（例如，調用代碼 | 重寫方法操作）時創建屬性被重寫 getter 方法的主體。
模板是可編輯的。 除了 Objective-C 表達式和註釋外，您還可以使用預定義變量，這些變量隨後將被擴展為相應值。
預定義變量將接收以下值：
${CALL_SUPER}   super 方法調用，對於 void 方法 — [super methodName]，帶或不帶形參； 對於其他方法 — return [super methodName]，帶或不帶形參
${RETURN_TYPE}   創建的方法的返回類型
${DEFAULT_RETURN_VALUE}   方法預設返回的值
${CONTAINING_CLASS}   在其中重寫方法的類的名稱
${IVAR_IS_AVAILABLE}   當屬性有實例變量 時，為 "true"
${IVAR}   實例變量的名稱
${SETTER_SEMANTICS}   在 ARC 模式下：已設置 copy 特性時，為 "copyArc"，否則為 "assign"
在非 ARC 模式下：已設置 copy 特性時，為 "copy"，已設置 retain 特性時，為 "retain"，否則為 "assign"
C 文件標頭模板（包含頁籤）中的所有預定義變量也可用（${USER}、${DATE} 等）

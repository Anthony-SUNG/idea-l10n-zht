在生成方法（例如，應用從用法創建方法意圖操作）時填充方法體。
此內置模板是可編輯的。 除了 Java 表達式和注釋外，您還可以使用預定義變量，這些變量隨後將被擴展為相應值。
預定義變量接收以下值：
${RETURN_TYPE}   創建方法的返回類型
${DEFAULT_RETURN_VALUE}   方法默認返回的值
${METHOD_NAME}   創建的方法的名稱
${CLASS_NAME}   創建方法的類的限定名稱
${SIMPLE_CLASS_NAME}   實現方法的類的非限定名稱
所有來自 文件頭 模板（包含選項卡）的預定義變量也可用。 （例如： ${USER}、 ${DATE} 等。）

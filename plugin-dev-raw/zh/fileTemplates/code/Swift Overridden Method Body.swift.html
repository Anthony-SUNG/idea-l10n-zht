在生成被重寫方法（例如，使用代碼 | 生成 | 重寫方法操作）時填充方法體。
模板是可編輯的。 除了 Swift 表達式和注釋外，您還可以使用預定義變量，這些變量隨後將被擴展為相應值。
預定義變量接收以下值：
${CALL_SUPER}   super 方法調用，用於 Void 方法：帶或不帶形參的 super.methodName()； 對於其他方法：帶或不帶形參的 return super.methodName()
${RETURN_TYPE}   創建的方法的返回類型
${DEFAULT_RETURN_VALUE}   方法默認返回的值
${SIGNATURE}   方法簽名
Swift 文件頭模板（包含選項卡）中的所有預定義變量也可用（例如 ${USER}、${DATE} 等）。
